## Overview👀

> 本组织是想聚集各届`成都七中`的同学们，汇聚优秀朋友，共同进步

+ 组员可提issue，然后邀请进入组织
+ issue 请注明:
  + 姓名
  + 毕业届数(or在读)
  + 专业方向
  + 个人特长(optional)
+ 作为GitHub开源组织，汇聚七中优秀的开发者和项目

进组织入口[点击](https://github.com/No-7-High-School/HelloSeveners/issues)留言🌲
